# 开篇场景写作技能

## 适用场景
- 小说开篇
- 新卷开篇
- 新篇章开始

## 写作要点

### 1. 黄金开篇原则
- 前500字必须抓住读者
- 抛出悬念或冲突
- 展示主角特点

### 2. 世界观展示
- 通过情节自然展示
- 避免大段说明
- 让读者自己探索

### 3. 建立期待
- 主角的目标和困境
- 金手指的暗示
- 后续剧情的伏笔

### 4. 常用句式
- "这一天，注定不平凡"
- "命运的齿轮，开始转动"
- "他不知道，这个决定将改变一切"
- "故事，要从这里说起"
- "一切的开始，源于……"

## 示例

```
林风从梦中惊醒，发现自己躺在熟悉的床上。

"又是这个梦……"他喃喃自语。

自从一个月前那场意外后，他总是做着同一个奇怪的梦。梦里有一片无尽的星空，还有一个看不清面容的老人。

他摇摇头，试图甩掉这些乱七八糟的想法。今天是他进入青云宗的第一天，可不能迟到。

然而他不知道的是，命运的齿轮已经开始转动。

那块他一直戴着的玉佩，此刻正微微发着光。
```

## 注意事项
- 开篇要有吸引力
- 不要堆砌设定
- 尽快让主角出场
