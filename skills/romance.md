# 感情场景写作技能

## 适用场景
- 感情发展
- 暧昧互动
- 告白时刻

## 写作要点

### 1. 氛围营造
- 环境描写烘托气氛
- 细节描写增加真实感
- 适当使用比喻和象征

### 2. 情感递进
- 从小细节开始
- 逐步深入情感
- 注意节奏控制

### 3. 心理描写
- 内心的纠结和挣扎
- 情感的细微变化
- 期待与忐忑

### 4. 常用句式
- "她的心跳不由得加快了几分"
- "空气中似乎弥漫着淡淡的甜意"
- "那一刻，时间仿佛静止了"
- "他的眼神，让她有些不知所措"
- "这份心意，她该如何回应？"

## 示例

```
夕阳的余晖洒在她的脸上，为她镀上一层柔和的金色。

他看着她，心中涌起一股说不清的情绪。

"你……"他开口，却又不知该说什么。

她抬起头，正好对上他的目光，脸颊不由得泛起一丝红晕。

两人就这样静静对视，谁也没有说话，却又仿佛说了很多。
```

## 注意事项
- 避免过于直白的表白
- 感情发展要有铺垫
- 注意人物性格的一致性
